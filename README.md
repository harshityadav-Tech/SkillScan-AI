# SkillScan AI

An AI-powered Resume Analyzer that predicts job roles, extracts skills, and calculates an ATS (Applicant Tracking System) score by comparing a resume with a job description.

## Features

- Predicts job category using Machine Learning
- Extracts technical skills from resumes
- Calculates ATS Score
- Shows matched skills
- Shows missing skills
- Progress bar for ATS Score
- Flask REST API backend
- HTML, CSS, JavaScript frontend

## Tech Stack

- Python
- Flask
- Scikit-learn
- Joblib
- HTML
- CSS
- JavaScript
